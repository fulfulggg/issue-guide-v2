# トヨタ「5回のなぜ」分析

## 概要

トヨタ生産方式の創始者・大野耐一氏が提唱した問題解決手法。表面的な原因ではなく、「なぜ？」を5回繰り返すことで真因（根本原因）にたどり着く。

## 有名な実例：機械停止トラブル

ある自動車工場で機械が突然停止するトラブルが発生した。

### 5回のなぜ

1. **なぜ機械が止まったのか？**  
   → ヒューズが切れたから

2. **なぜヒューズが切れたのか？**  
   → オーバーロード（過負荷）がかかったから

3. **なぜオーバーロードがかかったのか？**  
   → 軸受部の潤滑が十分でなかったから

4. **なぜ潤滑が十分でなかったのか？**  
   → 潤滑ポンプが十分に汲み上げていなかったから

5. **なぜポンプが十分に汲み上げていなかったのか？**  
   → ポンプの軸が摩耗していたから。切粉（金属粉）が入り込んでいたため

### 真因と対策

**真因**: ストレーナー（濾過器）が付いていないので切粉が内部に入り、潤滑ポンプが故障

**対策**: ストレーナーを取り付ける

### 教訓

もし表面的な「ヒューズ交換」だけで済ませていたら、数ヶ月後に同じトラブルが再発していた。5回のなぜによって真因に直接アプローチする解決策が見つかり、再発防止につながった。

## 別の実例：営業の問題

1. **なぜ新規顧客を獲得できていないのか？**  
   → 社長商談に持ち込めないから

2. **なぜ社長商談に持ち込めないのか？**  
   → 初回訪問から2回目訪問に進めないから

3. **なぜ2回目訪問に進めないのか？**  
   → 商品説明が長すぎて相手の課題をヒアリングできていないから

4. **なぜ商品説明が長すぎるのか？**  
   → 商品の特徴を全て伝えようとしているから

5. **なぜ全て伝えようとしているのか？**  
   → 相手のニーズを事前に把握せず、とりあえず全部説明すれば何か刺さると思っているから

**真因**: 事前のニーズ把握不足と、顧客中心ではなく商品中心の営業姿勢

**対策**: 初回訪問前に企業研究を行い、仮説を立ててから訪問する。商品説明は最小限にし、ヒアリング中心の対話に切り替える

## 5回のなぜの本質

- **回数は目安**: 必ずしも5回である必要はない。真因に到達するまで問い続けることが重要
- **再発防止**: 表面的な対処療法ではなく、根本原因を解決する
- **現場主義**: 机上の空論ではなく、現場・現物・現実（三現主義）に基づいて問う
- **粘り強さ**: 簡単な答えで満足せず、本質に届くまで問い続ける姿勢
